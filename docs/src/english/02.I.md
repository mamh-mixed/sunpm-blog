# /I/.

## 嘴形动作

1. 嘴微微张开，嘴唇放松，上齿和下齿之间约一个小指尖距离。
2. 舌尖抵下齿，舌硬腭抬起，舌位稍低，稍后，舌头肌肉松弛。
3. 嘴唇向两旁伸开，声带震动发出声音。
