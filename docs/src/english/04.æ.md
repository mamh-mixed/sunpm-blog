# /æ/
## 嘴形动作

1. 嘴唇向两侧尽量分开，嘴巴张开较大，上下齿之间大约可容纳两个手指的宽度。
2. 舌前部在发音过程中抬起，舌尖稍微接触下齿背。
3. 发音时逐渐压低舌头和下颚，尽可能夸张，震动声带，发出/æ/音。
